### **3. Exercice pratique**

**Objectif** : 
- Créez un document JSON décrivant une bibliothèque en ligne, contenant les éléments suivants :

   1. Une clé pour le **nom de la bibliothèque**.
   2. Un tableau d’objets pour les **livres disponibles**, chaque livre ayant :
      - Un titre,
      - Un auteur (prénom et nom),
      - Un genre,
      - Une disponibilité (booléen).
   3. Un tableau listant les **membres** de la bibliothèque, chaque membre ayant :
      - Un nom complet,
      - Une date d’inscription (chaîne au format AAAA-MM-JJ),
      - Une liste des livres empruntés (tableau d'id).
