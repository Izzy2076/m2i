# Exercice Positions NBA (Saison 2023-2024)

**Dataset** : [2023-2024 NBA Player Stats](https://www.kaggle.com/datasets/vivovinco/2023-2024-nba-player-stats)

## Découverte des Données

### Exercice 1 : Chargement et exploration initiale

**Objectif** : Charger le dataset et comprendre sa structure

À FAIRE :

1. Charger le fichier CSV (2023-2024_NBA_Player_Stats.csv)
2. Afficher les 10 premières lignes
3. Afficher les informations sur le dataset (.info())
4. Afficher les statistiques descriptives (.describe())
5. Vérifier les valeurs manquantes
6. Afficher les colonnes disponibles

### Exercice 2 : Nettoyage et sélection des features

**Objectif** : Préparer les données pour le machine learning

À FAIRE :

1. Supprimer les colonnes non pertinentes
2. Gérer les valeurs manquantes
3. Sélectionner les features numériques pour X
4. Définir y
5. Créer un train/test split
6. Afficher les dimensions de X_train, X_test, y_train, y_test

### Exercice 3 : Tester algorithmes

**Objectif** : Tester différents algorithmes

À FAIRE :

1. Créer une liste de modèles
2. Pour chaque modèle :

- Entraîner sur train set
- Prédire sur test set

3. Créer un DataFrame comparatif des scores
4. Visualiser les résultats avec un bar plot
