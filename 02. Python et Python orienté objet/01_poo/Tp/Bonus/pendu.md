# TP Python : Jeu du Pendu

1. La classe `JeuPendu` devra gérer :

   - Le mot à deviner
   - Les lettres trouvées
   - Les lettres déjà proposées
   - Le nombre de tentatives restantes

2. Fonctionnalités attendues :

   - Initialiser le jeu avec un mot aléatoire (depuis une liste interne ou un fichier)
   - Afficher l’état actuel du mot (ex : `_ e _ _ e`)
   - Demander une lettre à l’utilisateur
   - Mettre à jour les lettres trouvées ou le nombre d’erreurs
   - Afficher le message de victoire ou de défaite

3. Le programme principal (hors classe) devra :
   - Créer une instance du jeu
   - Lancer une boucle de jeu jusqu’à la victoire ou défaites
