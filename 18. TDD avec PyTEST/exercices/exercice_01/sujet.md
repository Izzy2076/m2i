## Exercice 01

Créez une fonction `valider_mot_de_passe(mdp)` qui retourne `True` si le mot de passe respecte ces critères :

- Au moins 8 caractères
- Contient au moins une majuscule
- Contient au moins un chiffre
- Contient au moins un caractère spécial (@, #, $, %, &)

Instructions TDD :

1. Commencez par écrire les tests (phase RED)
2. Implémentez le minimum pour passer les tests (phase GREEN)
3. Refactorisez votre code (phase REFACTOR)
